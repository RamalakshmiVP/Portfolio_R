💻 Projects



Asset Management System (Intern Project)

A full-stack web application developed during my internship using React.js, Node.js (Express), and PostgreSQL. This system handles asset tracking with admin/user role access and includes modules like Dashboard, Maintenance, Category, and Location.

The system was built for real-world IT usage where inventory and asset life cycle management is crucial. With clean UI and optimized backend, the project helped me understand team collaboration, Git version control, and deployment practices.

  • 🔐 Role-Based Authentication (Admin/User)
  • 📦 Category, Asset & Maintenance Management
  • 📊 Real-Time Dashboard with Summary Cards
  • 📤 Secure CRUD Operations with API integration
  • 🧑‍💻 Worked with mentor team in Agile environment

E-Voting System (UG Final Year Project)

A secure and efficient web-based voting platform built using HTML, CSS, and Python. The system ensures authenticated voting and maintains vote integrity, enabling safe digital elections.

The project addressed the growing need for online voting during remote or large-scale events. It offered me valuable experience in building user authentication and understanding the ethics of secure web development.

  • 🔒 Secure Login & Vote Casting System
  • 📁 Python-powered Backend Processing
  • 📱 Simple, Mobile-Responsive UI
  • 🗳️ Real-time vote status monitoring for admin
  • 🧠 First exposure to Python logic handling

Mushroom Classification using Decision Tree (Mini Project - PG 2nd Semester)

A machine learning-based application developed using Decision Tree Classifier to identify mushrooms as edible or poisonous. This mini project aimed to show practical implementation of classification algorithms.

It was part of the core syllabus and motivated me to explore how machine learning can solve real-life biological and food safety problems. I used datasets with attributes like color, odor, and gill shape.

  • 🌿 Dataset with over 20 mushroom features
  • 📈 Decision Tree model trained & tested
  • 📊 Confusion matrix & accuracy graph plotted
  • 🔍 Feature importance & prediction output analysis
  • 🤖 Initiated my interest in Python-based AI projects
